== DESCRIPTION ==
zeStatix is counter clicks for the specified HTML elements.

Features:
- The ability to select an element on the page;
- No additional servers are used;
- No cookies used.

Statistics:
- Number of element impressions on the page;
- Number of clicks;
- Click time;
- Country, city;
- IP;
- Device;
- Monitor width;
- Browser width.
